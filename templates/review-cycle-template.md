## 1. 사이클 개요

### 1.1 주기
- **공식 평가**: 반기 (1월, 7월)
- **중간 체크인**: 분기
- **1on1**: 주간 (권장) / 격주 (최소)

### 1.2 참여 대상
- 전 구성원 (입사 3개월 이상)
- 신입: 첫 사이클은 Calibration 제외

---

## 2. 평가 축

### 축 1: What (성과)
- 가중치: 50%
- 평가 방법: OKR 달성도 + 정성 평가
- 근거: 구체적 성과 사례 최소 3개

### 축 2: How (역량/행동)
- 가중치: 40%
- 평가 방법: 레벨별 기대 역량 루브릭
- 근거: 행동 관찰 사례

### 축 3: Potential (잠재력) — 승급 후보만
- 가중치: 10% (승급 논의 시)
- 평가 방법: 차기 레벨 역량 증거

---

## 3. 등급 체계

### 5단계 등급

| 등급 | 명칭 | 분포 가이드 | 의미 |
|------|------|-------------|------|
| 5 | Far Exceeds | ~5% | 모든 기대를 크게 초과 |
| 4 | Exceeds | ~20% | 기대 초과, 승급 후보 |
| 3 | Meets | ~60% | 기대 충족 |
| 2 | Below | ~10% | 기대 미달, 개선 필요 |
| 1 | Far Below | ~5% | 심각한 미달, PIP 대상 |

*분포 가이드는 참고용, 강제 배분 아님*

## 4. 사이클 구성 요소

### 4.1 Goal Setting (목표 설정)
- **시점**: 사이클 시작 2주 내
- **주체**: 구성원 → 매니저 합의
- **도구**: OKR 템플릿
- **산출**: 합의된 목표 문서

### 4.2 Mid-cycle Check-in
- **시점**: 사이클 중간
- **주체**: 매니저-구성원 1:1
- **내용**: 진행 상황, 장애물, 목표 조정
- **산출**: Check-in 노트

### 4.3 Self Review
- **시점**: 사이클 말 진입 4주 전
- **주체**: 구성원 본인
- **양식**: 첨부 ([Self Review Form])
- **마감**: 2주 내

### 4.4 Peer Review (선택)
- **시점**: Self Review와 병행
- **주체**: 프로젝트 동료 2~5명
- **양식**: 첨부 ([Peer Review Form])
- **마감**: 2주 내

### 4.5 Manager Review
- **시점**: Self/Peer 마감 후 2주
- **주체**: 직속 매니저
- **양식**: 첨부 ([Manager Review Form])
- **산출**: 등급 + 근거 + 개발 계획

### 4.6 Calibration
- **시점**: Manager Review 마감 후 1주
- **참여자**: 직군 내 매니저 + HR Business Partner
- **프로세스**: (아래 상세)
- **산출**: 최종 등급 (조정 근거 기록)

### 4.7 평가 면담
- **시점**: Calibration 후 1주
- **주체**: 매니저 ↔ 구성원
- **내용**: 피드백 + 개발 계획 합의
- **주의**: **이 시점에는 보상 언급 금지**

### 4.8 보상 결정
- **시점**: 평가 면담 후 2주
- **주체**: HR + 경영진
- **근거**: 평가 등급 + 예산 + 시장 데이터
- **산출**: 개인별 조정안

### 4.9 보상 면담
- **시점**: 보상 결정 후 1주
- **주체**: 매니저 ↔ 구성원
- **내용**: 조정 결과 통보 + 근거 설명

## 6. Calibration 프로세스

### 6.1 참여자
- 직군/부서 내 매니저 전원
- HR Business Partner (퍼실리테이터)
- 필요 시 상위 리더

### 6.2 사전 준비
- 각 매니저: 팀원 잠정 등급 제출
- HR: 전체 분포 집계
- HR: 특이 케이스 식별 (분포 이상, 이견 예상)

### 6.3 회의 진행 (2~3시간)

1. **분포 검토** (15분)
   - 현재 잠정 분포 확인
   - 분포 가이드 대비 이상 확인

2. **High Performer 검토** (45분)
   - Exceeds/Far Exceeds 제안자 각각 검토
   - 근거 발표 → 동료 매니저 질문 → 합의

3. **Low Performer 검토** (30분)
   - Below/Far Below 제안자 검토
   - 개선 계획 논의

4. **Borderline 검토** (45분)
   - Meets ↔ Exceeds 경계
   - Meets ↔ Below 경계

5. **최종 분포 확정** (15분)
   - 조정 결과 종합
   - 다음 단계 확인

### 6.4 기록
- 각 구성원의 최종 등급
- 조정이 있었다면 근거
- 회의록 (조직 내 제한 공개)

### 6.5 주의사항
- **순위 매기기 아님**
- 근거 없는 하향 조정 금지
- 개인 비난 금지 (사실 기반 논의)

## 7. 평가 ↔ 보상 연결

### 7.1 연결 방식
**분리형 (권장)**:
- 평가 면담 (봄) → 성장 중심 대화
- 보상 면담 (가을) → 결정 통보
- 사이에 HR 조정

### 7.2 인상률 가이드

| 등급 | 인상률 범위 | 비고 |
|------|-------------|------|
| Far Exceeds | Base +8~15% | 승급 동반 가능 |
| Exceeds | Base +5~8% | 승급 후보 |
| Meets | Base +3~5% | 시장 조정 포함 |
| Below | Base 0~2% | PIP 논의 |
| Far Below | Base 0% 또는 동결 | PIP 필수 |

*인상률은 예산 제약과 페이밴드 위치에 따라 조정*
