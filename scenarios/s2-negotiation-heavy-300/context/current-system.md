# Current System — s2

## Compensation

- **Pay Band**: L1~L8 (엔지니어) / L1~L6 (비엔지니어) 밴드 있음
  - 그러나 **이탈자(out-of-band) 40% 이상** — 특히 L4~L6 구간
  - 이탈 사유 대부분: 스카우트 경쟁 / 역오퍼 대응
- **인상**: 연 1회 3월. 성과등급(S/A/B/C/D) × 시장 조정
  - **S 비중 35%, A 비중 48%** → 사실상 대부분 상단 등급
- **사인온 보너스**: 채용 담당자 재량, 범위 공식화 없음
- **Equity**: IPO 기대주. Refresh grant 있음 (주관적 판단)

## Performance

- **리뷰 사이클**: 연 1회 + 분기 self-check
- **평가자**: 직속 매니저 단독 (2차 리뷰 없음)
- **Calibration**: **없음** — 각 매니저가 자기 팀 등급 독립 배정
- **360 리뷰**: Director 이상에게만 시행 (10명)
- **1on1**: 가이드는 있으나 실행률 팀마다 상이

## 문서·정책

- HR 위키 존재, 업데이트는 분기 1회 수준
- 밴드 수치 비공개 (내부 Director 이상만 열람)
- 평가 기준·분포 정책 사내 공지 없음
