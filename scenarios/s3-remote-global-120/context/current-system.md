# Current System — s3

## Compensation

- **초기 정책**: "Location-agnostic" (지역 무관 동일 급여) — 공식 문서화
- **현실**: 2023년부터 **신규 입사자는 로컬 조정** 내부 적용 (시장별 P50 기준)
  - 기존 입사자는 그대로 → **같은 역할·같은 레벨인데 위치에 따라 최대 2배 격차**
- **통화**: 계약은 USD 기본, 로컬 통화 지급 옵션
- **인상**: 연 1회, 지역별 시장 데이터 반영하나 **어떤 소스 쓰는지 비공개**
- **Equity**:
  - RSU 부여 (Series B 후)
  - 국가별 세법 적용 불일치 → 실수령액 예측 불가 사례 빈번

## Performance

- **리뷰**: 반기 1회 (정량 목표 + 360)
- **시차 문제**: 라이브 Calibration 불가 → 비동기 slack/docs 기반
- **피드백**: 분기 1on1 + Loom 비동기 리뷰
- **타임존 격차**로 인해 서구권 매니저가 아시아·아프리카 IC 가시성 낮음

## 문서·정책

- Public handbook 일부 공개 (GitLab 스타일)
- 밴드·지역 조정 계수 비공개
- Compensation philosophy 2년 전 블로그 글에만 존재, 내부 위키에 업데이트 안 됨
